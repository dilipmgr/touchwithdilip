By DKT. Sep 15,2020

Introduction:-Python is an interpreted high-level general-purpose programming language. Python's design philosophy emphasizes code readability with its notable use of significant indentation. Its language constructs as well as its object-oriented approach aim to help programmers write clear, logical code for small and large-scale projects.

1.Comments are non-executable statements in a program.

2.An identifier is a user defined name given to a variable or a constant in a program.

3.Process of identifying and removing errors from a computer program is called debugging.

4.Trying to use a variable that has not been assigned a value gives an error.

5.There are several data types in Python â€” integer, boolean, float, complex, string, list, tuple, sets, None and dictionary.

6.Operators are constructs that manipulate the value of operands. Operators may be unary or binary.

7.An expression is a combination of values, variables, and operators.

8.Python has input() function for taking user input.

9.Python has print() function to output data to a standard output device.

10.The if statement is used for decision making.

11Looping allows sections of code to be executed repeatedly under some condition.

12.for statement can be used to iterate over a range of values or a sequence.

13.The statements within the body of for loop are executed till the range of values is exhausted.

An ordered set of instructions or commands to be executed by a computer is called a program. The language used to specify those set of instructions to the computer is called a programming language for example Python, C, C++, Java, etc. This chapter gives a brief overview of Python programming language. Python is a very popular and easy to learn programming language, created by Guido van Rossum in 1991. It is used in a variety of fields, including software development, web development, scientific computing, big data and Artificial Intelligence.
